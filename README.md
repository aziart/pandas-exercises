A structured collection of **pandas practice exercises** created to strengthen data manipulation skills through hands-on repetition. This repository serves as a personal practice workspace — a place to explore, experiment, and reinforce pandas concepts, rather than provide finalized or curated solutions.
The exercises are based on:
[Guilherme Samora](https://github.com/guipsamora)'s [repository](https://github.com/guipsamora/pandas_exercises)
